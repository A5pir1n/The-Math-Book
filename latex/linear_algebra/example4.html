Basic Logical Quantifiers

Definition 1. A math statement is a statement that is either True or False. For example:

  • "An elephant is an animal" is True.

  • "A dandelion is an animal" is False.

  • "1 + 1 = 2" is True.

  • "3 ≤ 2" is False.

Definition 2. "¬X" is the negation of statement X (namely "not X"). The statement "¬X" is True iff (if and only if) statement X is False. For example:

  • "An elephant is an animal", which is True.
    Negation: "An elephant is not an animal", which is False.

  • "A dandelion is an animal", which is False.
    Negation: "A dandelion is not an animal", which is True.

Definition 3. Conjunction (and) & Alternative (or)
The statement "X ∧ Y" (read as "X and Y") is True iff X and Y are both True.
The statement "X ∨ Y" (read as "X or Y") is True iff at least one of X, Y are True

Definition 4. The statement "X ⇒ Y" (read as "statement X implies statement Y") means that "if statement X is True, then statement Y is also True".

Remark 1. The relationship of X, Y, and X ⇒ Y is shown in the following "truth table":

X Y X ⇒ Y
T T T
T F F
F T T
F F T

Remark 2. Note that if X is False, then X ⇒ Y is always True regardless of Y. Therefore, when we want to prove X ⇒ Y, we only need to assume X is True, and then prove that Y is True under such assumption.

Remark 3. Note that X ⇒ Y is False only when X is True and Y is False. Therefore, the negation of "X ⇒ Y" is "X ∧ (¬Y)", namely "X and not Y".

Remark 4. The statement "X ⇒ Y" is equivalent to "¬Y ⇒ ¬X". The latter is called a contra-positive of "X ⇒ Y". Therefore, when we want to prove X ⇒ Y, we can instead prove its contra-positive.

X Y X ⇒ Y ¬Y ¬X ¬Y ⇒ ¬X

T F F T F F
F T T F T T
F F T T T T

Definition 5. The statement "X ⇔ Y" (read as "X iff Y" or "X is equivalent to Y") is True iff "X ⇒ Y and Y ⇒ X" is True.

Remark 5. By definition, when we want to prove X ⇔ Y, we need to prove X ⇒ Y(⇒, namely "forward direction") and Y ⇒ X(⇐, namely "backward direction").

Definition 6. Universal quantifier: ∀
The statement "∀ (x in some range X), (statement Z)" is True iff for all objects x in the specified range, statement Z is True. For example:

  • "∀ natural number x, x ≥ 0" is True because all natural numbers are greater or equal to 0.

Definition 7. Existential quantifier: ∃
The statement "∃ (x in some range X), such that (statement Z)" is True iff there exists at least one object x in the specified range such that statement Z is True. For example:

  • "∃ natural number x, s.t. x > 2" is True because x = 3 is a natural number that makes x > 2.

Remark 6. Negations of universal and existential statements

  • The negation of "∀ (x in range X), (statement Z)" is "∃ (x in range X), s.t. (¬Z)". Therefore, to prove that a universal statement is False, we need to find a counter example. For example:
    The statement "∀ elephants on Mars, that elephant is purple" is True because we cannot find any counter examples since there are no elephants on Mars (at least for now).

  • The negation of "∃ (x in range X), such that (statement Z)" is "∀ (x in range X), (¬Z)".

    For example, in order to prove that "∃ elephant on Earth, s.t. that elephant is purple" is False, we need to check that all elephants on Earth are not purple, namely to prove "∀ elephant on Earth, that elephant is not purple".

Set

A set is a list of distinct objects enclosed by curly brackets "{}". The set itself is usually denoted with capital letters. For example, A = {1, 2, 3, 4} is the set containing 1, 2, 3, and 4.

Definition 8. If an object is in a set, then it is an element of that set, denoted with the symbol "∈". In the previous example, 2 is an element of A, thus we say "2 ∈ A". The symbol "∉" is used to denote that an object is not an element of a set. In the previous example, 0 is not an element of A, thus we say "0 ∉ A".

Remark 7. By convention, the following set of numbers are denoted with special symbols:

  • ℕ is the set containing all naturally numbers.

  • ℤ is the set containing all integers.

  • ℚ is the set containing all rational numbers.

  • ℝ is the set containing all real numbers.

Definition 9. A set containing no elements is a called a trivial set or an empty set, denoted as ∅ = {}.

Definition 10. If all elements of a set X are also element of another set Y, then we say X is a subset of Y, denoted as "X ⊆ Y". If any element of X is not element of Y, then X is not a subset of Y, denoted as X ⊈ Y. Namely: X ⊆ Y ⇔ (∀x∈X,x∈Y)

Definition 11. X and Y represent the same set iff X and Y are subset of each other. Namely, X = Y⇔ $(X\subseteq Y \hspace{2} and \hspace{3} Y \subseteq X)$.

Definition 12. set operators
For any sets X and Y:

  • X ∪ Y (read as "X union Y") is the set containing all elements that is element of X OR element of Y.

  • X ∩ Y (read as "X intersect Y") is the set containing all elements that is element of X AND element of Y.

  • X − Y (read as "X minus Y") is the set containing all elements that is element of X and NOT element of Y.

Remark 8. Set-building notation
The set X = {x in some range | statement about x} (":" can be used to replace "|") contains all objects x in the specified range that makes the specified statement True.
For example, the set $X=\{x \in \mathbb{N} : \exists k \in \mathbb{N}, \hspace{2 } s.t. \hspace{3} x=2k\}$ is the set containing all natural numbers that are even.

Functions

Definition 13. Intuitively, a function f from a set A to set B, written as f : A → B, maps elements a ∈ A to a single element f(a) ∈ B. In this case, A is called the domain of f, and B is called the co-domain of f.
The set A × B is the set of all ordered pairs (a,b), where a ∈ A and b ∈ B. Formally speaking, f is a subset of A × B. Thus, (a,b) ∈ f is another way to represent f(a) = b.

Remark 9. A function f : A → B is well-defined iff it satisfies both of the following:

  • $\forall \hspace{2} a \in A, \exists \hspace{2} b \in B \hspace{3} s.t. \hspace{3} (a, b) \in f$, namely all elements of A are mapped to some elements of B by f.

  • $\forall \hspace{2} b' \in B, b' \ne b \Longrightarrow (a, b') \notin f$, namely each element in A is mapped to exactly one element in B by f.

All functions must be well-defined in their specific domains and co-domains. Therefore, in future discussion we can assume all functions are well-defined, and when we want to defined a new function, we need to make sure the it is well-defined.

Definition 14. A function f : A → B is an injection (injective/one-to-one) iff $\forall \hspace{2} a, a' \in A, a \ne a' \Longrightarrow f(a) \ne f(a')$, namely each element in A is mapped to a unique element in B by f.

Definition 15. A function f : A → B is a surjection (surjective/onto) iff $\forall \hspace{2} b \in B, \exists \hspace{2} a \in A \hspace{3} s.t. \hspace{3} f(a) = b$, namely all elements in B are mapped to by f.

Definition 16. A function f : A → B is a bijection iff it is both injective and surjective.
