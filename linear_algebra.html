Equivalence Theorems

Basis

The following statements are equivalent:

  • A set B is a basis for a vector space V

  • The spanning set of the vector space V does not have strict subset that is also a spanning set

  • The set B does not have any strict superset that is also linearly independent.

Norms and normed vector spaces

For a vector space V over field F with element x ‚àà V, a norm of V is a function $\norm{x}: V \rightarrow [0,\infty)$ satisfying:

  1. $\norm{x} = 0$ if and only if x = 0.
