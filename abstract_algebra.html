Rings

One

A ring is a set K, together with two operations (x,y) → (x + y) and (x,y) → xy satisfying:

  1. K is a commutative group under the operation (x,y) → (x + y) (K is a commutative group under addition);

  2. (xy)z = x(yz) (multiplication is associative);

  3. x(y+z) = xy + xz; (y+z)x = yx + zx (distributive laws hold.

If xy = yx for all x and y in K, we say that the ring K is commutative. If there is an element 1 in K such that 1x = x1 = x for each x, K is said to be a ringwithidentity, and 1 is called the identity for K.
