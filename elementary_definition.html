Permutation

A sequence (k1,k2,⋯kn) of positive integers not exceeding n, with the property that no two of the ki are equal, is called a permutation of degree n.

Since a finite sequence, or n-tuple, is a function defined on the first n positive integers, a permutation of degree n may be defined as a one-to-one function from the set {1, 2, ⋯n} onto it self. Such a function σ corresponds to the n-tuple (σ1,σ2,⋯,σn) ad is thus simply a rule for ordering 1, 2, ⋯, n in some well-defined way.

All possible number of permutation of degree n is n!.

Elementary combinatorics

If we only consider rearrangements of k elements taken a given set of size n, we call them k-permutations of n, which are different ordered arrangements of a k-element subset of an n-set. They are also known as partial permutations or as sequences without repetition.

Another closely related concept is combination, the element of which is not ordered.
